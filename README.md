# DrawPlayingCards

Simple card game, created as programming exercise in C# using WPF with additional option to save current card deck and drawn cards to txt file. 

#### Mechanics:  

-	Create card deck 
-	Shuffle cards
-	Sort cards
-	Drag and drop cards
-	Save the status of your cards in file txt
